# UMTX 2 v2.1.1 — fPKG Release

## 🎮 What's New

This release introduces **PS5 fPKG installation support**, allowing you to install UMTX 2 as a native PS5 application on your jailbroken console.

### Added

- **PS5 fPKG Package** (`PPSX43000-KemalSanli UMTX 2.pkg`) — 9.39 MB
  - Install directly on your PS5 home screen
  - One-click access to UMTX 2 host
  - No browser bookmarks needed
  - Opens directly to `https://kemalsanli.github.io/umtx2/`

### Installation Methods

**Option A: Browser (recommended)**
- Visit the host URL directly in your PS5 browser
- No installation required

**Option B: fPKG Install (new!)**
1. Download `PPSX43000-KemalSanli UMTX 2.pkg` from this release
2. Install via etaHEN package installer, FTP, or WebDAV
3. Launch from your PS5 home screen
